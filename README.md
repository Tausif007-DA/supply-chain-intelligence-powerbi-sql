# supply-chain-intelligence-powerbi-excel
This project focuses on analyzing supply chain operations using Microsoft Excel and Power BI to identify performance gaps and improve operational efficiency.  The objective was to transform raw supply chain data into meaningful business insights through KPI tracking, data modeling, and interactive dashboard visualization.
🎯 Business Objective

The organization was facing challenges such as:

High percentage of delayed deliveries

Inconsistent supplier performance

Inventory imbalance across warehouses

Limited visibility into regional revenue trends

This project aims to address these issues by building a structured analytics solution.

🛠 Tools Used

Microsoft Excel – Data cleaning, preprocessing, and KPI calculations

Power BI – Data modeling, DAX measures, dashboard development

📊 Key KPIs Developed

Total Revenue

Profit Margin

Late Delivery Percentage

Average Delivery Time

Supplier Risk Score

Inventory Level by Warehouse

Regional Performance Analysis

📈 Key Insights

55% of total orders were delivered late

High-risk suppliers significantly impacted delivery performance

Revenue is concentrated in specific regions

Inventory imbalance is affecting supply chain consistency

💡 Business Recommendations

Improving supplier monitoring and optimizing high-risk vendors can reduce delays and improve efficiency. Inventory balancing and focusing on high-performing regions can enhance profitability and operational performance.

🚀 Future Enhancements

Add predictive delivery analysis

Implement inventory forecasting

Connect with real-time data sources
